Exercise 6.1.
- Question: What happens if users choose "Cancel"?
- Answer: The showConfirmDialog method returns the integer value 2 (JOptionPane.CANCEL_OPTION).

- Question: How to customize the options to users (e.g., only "Yes" and "No")?
- Answer: We can use the showConfirmDialog method with the "optionType" parameter set to JOptionPane.YES_NO_OPTION. For full customization (like "I do" and "I don't"), we should use JOptionPane.showOptionDialog.
